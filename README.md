# PAGAPP

PAGAPP es una aplicación móvil diseñada para facilitar la gestión de gastos compartidos entre amigos. Permite registrar gastos, dividir costos entre participantes, gestionar deudas, registrar pagos y visualizar el historial de ingresos y egresos.

La aplicación busca simplificar la administración de gastos grupales en situaciones como viajes, salidas a restaurantes o cualquier actividad compartida.

---

# Objetivo del Proyecto

El objetivo de PAGAPP es desarrollar una solución digital que permita a los usuarios:

- Registrar gastos compartidos
- Dividir gastos entre amigos
- Gestionar deudas pendientes
- Registrar pagos entre usuarios
- Recibir notificaciones de transacciones
- Consultar el historial de ingresos y egresos

---

# Funcionalidades Principales

Las funcionalidades principales del sistema incluyen:

## Registro de usuario
Los usuarios pueden crear una cuenta para acceder a la aplicación.

## Inicio de sesión
Permite a los usuarios autenticarse en el sistema para acceder a su información financiera.

## Panel principal
El usuario puede visualizar:
- Balance general
- Deudas pendientes
- Actividad reciente

## Registro de gastos compartidos
Los usuarios pueden registrar gastos e indicar los participantes del gasto.

## Gestión de deudas
El sistema calcula automáticamente cuánto debe pagar cada participante.

## Registro de pagos
Los usuarios pueden registrar pagos para saldar deudas.

## Historial de transacciones
Permite consultar los movimientos financieros realizados dentro de la aplicación.

## Notificaciones
Los usuarios reciben notificaciones cuando:
- se registra un gasto
- se realiza un pago
- se actualiza el estado de una deuda

## Detección de amigos cercanos
La aplicación puede usar la ubicación del dispositivo para detectar amigos cercanos y facilitar el registro de gastos compartidos.

---

# Arquitectura del Sistema

El sistema se compone de los siguientes elementos principales:

- **Usuarios**: personas que utilizan la aplicación.
- **Gastos (Expense)**: registros de gastos compartidos.
- **Deudas (Debt)**: obligaciones generadas a partir de los gastos.
- **Pagos (Payment)**: transacciones que permiten saldar deudas.
- **Notificaciones (Notification)**: avisos generados por el sistema.
- **Historial (HistoryEntry)**: registro de ingresos y egresos.
- **Ubicación (LocationContext)**: información utilizada para detectar amigos cercanos.
